Notes & Edge Cases
Multi-page bills
Claude reads all pages. HST and the Ontario Electricity Rebate often appear on page 2 or 3. The system captures these even when not on page 1.
CR (credit) notation
On electricity bills, the Ontario Electricity Rebate shows as "CR." The system treats any CR line as a value to subtract, not add.
General Service <50kW
Smaller electricity accounts don't have a separate demand structure. Demand and demandCost return null for these accounts.
Transformer Allowance
Appears only on accounts with customer-owned transformers. When present it is subtracted from demandCost.
Estimated reads
Bills with estimated readings are flagged (estimation = "Yes") so they can be treated as provisional. Prior estimated rows should be manually corrected when the actual read arrives.
Property ID extraction
System checks billing name block, service address, then filename. Supported formats include MTCC, TSCC, YCC, YRCC, YRSCC, PSCC, PCC — and others may exist across Ontario.
