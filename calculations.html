Notes & Edge Cases
Multi-page bills
Claude reads all pages. HST and the Ontario Electricity Rebate often appear on page 2 or 3 of electricity bills, not page 1. The system is designed to capture these even when they are not on the first page.
CR (credit) notation
On electricity bills, the Ontario Electricity Rebate is shown as a credit with "CR" next to the amount. The system treats any CR line as a value to subtract, not add.
General Service <50kW
Smaller electricity accounts do not have a separate demand charge structure. For these accounts, demand and demandCost return null, and demandCost captures the Delivery total only if applicable.
Transformer Allowance
This credit appears only on accounts with customer-owned transformers. When present it is subtracted from demandCost. The system checks for it but does not assume it will be there.
Estimated reads
Bills with estimated readings are flagged in the sheet (estimation = "Yes") so they can be treated as provisional. When the actual read arrives the following month, the prior estimated row should be manually corrected.
Property ID extraction
The system looks for the condo corp number in the billing name block and service address. If not found there, it checks the PDF filename. Supported formats include MTCC, TSCC, YCC, YRCC, YRSCC, PSCC, PCC — and others may exist across Ontario.
